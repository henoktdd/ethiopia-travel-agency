# Discover the Wonders of Ethiopia — Travel Agency
A semantic HTML travel agency project for freeCodeCamp featuring tour packages and top Ethiopian travel itineraries.

## Features
* **Semantic HTML:** Clean HTML5 markup (`<figure>`, `<figcaption>`, lists and headings).
* **Featured Destinations:** Highlights top travel spots including Axum, the Simien Mountains and Lalibela.
* **Tour Offerings:** Direct links for group travels and private tour options.
* **SEO Ready:** Includes meta descriptions for search engines.
